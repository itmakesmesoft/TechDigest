## API

| 옵션       | 타입                                    | 기본값    | 설명                                            |
| ---------- | --------------------------------------- | --------- | ----------------------------------------------- |
| `type`     | `default`\| `info`\| `warning`\|`error` | `default` | 콜아웃 타입                                     |
| `emoji`    | `string` \| `ReactNode`                 |           | 콜아웃에 보여질 아이콘. 커스텀 아이콘 사용 가능 |
| `children` | `ReactNode`                             |           | 콜아웃 내용                                     |
